## Urban Routes Application – Functional Testing

### Description
Performed functional testing for an urban transportation application.

### Project Achievements

- Designed and executed manual test cases to validate key application functionalities, including the complete “Request a Taxi” workflow and interaction with the “Reserve” button.

- Created checklists to ensure proper visualization and functionality of user interface (UI) elements such as input fields, buttons, error messages, and dynamic components.

- Performed cross-browser compatibility testing (Chrome, Firefox), validating application behavior, responsive design, and usability across different environments.

- Identified and documented defects in Jira using structured bug reports, including reproduction steps, priority level, test environment, and screenshots as supporting evidence.

- Contributed to improving the user experience by identifying accessibility issues and incomplete user flows during exploratory testing.

- Participated in a simulated Agile collaborative environment, completing sprint deliverables while applying QA best practices.

**Date:** January 2026  
**Tools Used:** Figma, Jira, Chrome, Firefox
